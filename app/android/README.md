# Android版 - maimai随机选歌工具

## 开发环境设置

1. 确保已安装Flutter SDK
2. 安装依赖：
```bash
flutter pub get
```

3. 运行应用：
```bash
flutter run
```

4. 打包APK：
```bash
flutter build apk
```

## 功能说明

### 选歌条件
- 难度选择：Easy ~ Re:Master
- 歌曲类型：标准(STD) / DX
- 流派筛选
- 等级范围：支持整数(14)、加号(14+)、小数(14.5)输入
- 随机次数：可重复抽取

### 等级输入规则
- 整数输入 (如 14): 匹配 14.0 ~ 14.5 (广义等级)
- 带加号输入 (如 14+): 匹配 14.6 ~ 14.9
- 小数输入 (如 14.5): 精确匹配 14.45 ~ 14.55
